# Paddle Game using Flask

A simple web-based paddle game developed using Python and Flask. The application runs in a browser and demonstrates basic game mechanics with server-side routing.

## Features
• Interactive paddle game interface  
• Built using Flask backend and HTML/CSS/JavaScript frontend  
• Real-time game interaction in browser  

## Tech Stack
Python, Flask, HTML, CSS, JavaScript

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open browser at:

http://127.0.0.1:5000
